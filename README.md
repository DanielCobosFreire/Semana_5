El estudiante deberá actualizar el proyecto web trabajado en la Semana 4, incorporando JavaScript para crear una funcionalidad dinámica. La página deberá permitir registrar información desde un formulario y mostrarla en pantalla sin recargar el navegador. Esta funcionalidad puede aplicarse a productos, servicios, citas, solicitudes, tareas, clientes o cualquier elemento relacionado con el tema del proyecto.
